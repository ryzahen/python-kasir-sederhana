# python-kasir-sederhana
Program kasir sederhana berbasis Python yang berjalan di terminal untuk menghitung total belanja, diskon, dan kembalian.

Program ini adalah contoh aplikasi kasir sederhana yang dibuat menggunakan bahasa Python.  
Program berjalan di terminal / command prompt dan dapat digunakan untuk menghitung total belanja, diskon, pembayaran, dan kembalian.

Program ini dibuat sebagai latihan dasar pemrograman Python.

----

## Fitur Program

Beberapa fitur yang tersedia dalam program ini:

- Menampilkan daftar barang
- Memilih barang berdasarkan kode
- Menginput jumlah barang yang dibeli
- Menghitung total belanja
- Memberikan diskon sesuai jumlah belanja
- Menghitung kembalian dari pembayaran
- Validasi input agar hanya menerima angka
- Validasi jika uang pembayaran kurang

----

## Daftar Barang

Barang yang tersedia di program ini:

| Kode | Barang | Harga |
|-----|------|------|
| 1 | Baju | 10000 |
| 2 | Celana | 30000 |
| 3 | Sepatu | 25000 |
| 4 | Sandal | 15000 |
